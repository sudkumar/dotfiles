# Dotfiles

### Install

Install xcode dev tools in you are on mac.
```bash
xcode-select --install
``` 

Then clone to `~/.dotfiles` and install

```bash
git clone https://github.com/sudkumar/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```
### Backup files
```bash
 install/backup.sh
```
#### VIM Installation
```bash
vim +PlugInstall
```
