# Dotfiles

## Installation

### Backup files
```bash
 install/backup.sh
```

### Install

Install xcode dev tools in you are on mac.
```bash
xcode-select --install
``` 

Then clone to `~/.dotfiles` and install

```bash
git clone https://github.com/nicknisi/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

#### VIM Installation

```bash
nvim +PlugInstall
```
